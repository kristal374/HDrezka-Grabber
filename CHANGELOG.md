# Change Log

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/)
and this project adheres to [Semantic Versioning](http://semver.org/).

## [[1.0.0](https://github.com/kristal374/HDrezka-Grabber/releases/tag/v1.0.0)] - 2026-02-24

Крупнейшее обновление! Всё расширение полностью переписано с нуля, было написано свыше 25 тысяч новых строк кода, при этом процесс сборки позволил сократить размер расширения в два раза по сравнению предыдущей версией.

### Added

- Добавлена возможность загрузки субтитров.
- Добавлена более гибкое управление загрузкой сериалов, теперь можно выбирать диапазон серий для загрузки.
- Добавлена возможность указать пользовательский шаблон имени файла для загрузки.
- Добавлено определение размера загружаемого качества.
- Добавлена параллельная загрузка нескольких файлов.
- Добавлена возможность загрузки нескольких фильмов/сериалов одновременно.
- Добавлена история загрузок (пока невозможно просмотреть, интерфейс будет доступен в будущих релизах).
- Добавлены настройки для управления расширением.
- Добавлены экраны-заглушки для уведомления пользователей о пригодности страницы для загрузки.
- Добавлены новые темы для расширения (тёмная, светлая).
- Наконец добавлен выбор озвучки у фильмов (ранее автоматически подтягивался со страницы).
- Добавлена собственная система логирования, сообщать об ошибках станет проще.
- Закинул пачку новых багов, так как старые уже надоели.

### Changed

- Весь проект полностью переписан с нуля и был переведён на React.
- Портировано на Firefox, дополнительно опубликовано в firefox и edge Store.
- Изменён механизм загрузки файлов, теперь файлы загружаются через browser API напрямую на диск.
- Более не нужно находиться на странице фильма для загрузки файлов, теперь инициатором выступает само расширение.
- Теперь имена серий/сезонов в попапе подтягиваются полностью.
- Теперь прогресс загрузки в попапе дополнительно показывает общее число файлов к загруженным. Станет проще понимать, на каком этапе находится загрузка.
- Переработано выпадающее меню в попапе, теперь поддерживается отображение изображений (флаги, иконка премиума), отображение размера видео, добавлен поиск. Однако теперь выпадающее меню не может быть больше окна попапа.
- Переработана система уведомлений в попапе.

### Fixed

- Исправлены все ошибки старой версии, для того чтоб было место для новых в новом релизе!

## [[0.2-beta](https://github.com/kristal374/HDrezka-Grabber/releases/tag/v0.2)] - 2023-09-29

Релиз содержит в основном правки багов.

### Added

- Добавлено подтягивание названия озвучки (в тех случаях, когда это возможно) взамен надписи "Неизвестная озвучка".

### Changed

- Проведён лёгкий рефакторинг кода.

### Fixed

- Исправлена ошибка, при которой в редких случаях во время скачивания сериала прерывалась загрузка. Это было связано с

- Исправлена логика последовательной загрузки серий в тех случаях, когда загрузка начинается не с первой серии.
- Исправлена ошибка с отображением некорректного списка серий (из другого сезона) при смене озвучки.
